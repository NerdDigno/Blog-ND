Instruções para Edição do Site

Estrutura de Arquivos

O site Nerd Digno foi criado com uma estrutura simples para facilitar a edição:

  • index.html - Página inicial do site
  • animes.html, games.html, etc. - Páginas de categorias
  • post-anime1.html, post-game1.html, etc. - Páginas de posts individuais
  • css/style.css - Arquivo de estilos do site
  • js/script.js - Arquivo JavaScript com funcionalidades básicas
  • images/ - Pasta para armazenar as imagens do site

Como Editar o Conteúdo

Para editar o conteúdo do site, siga estas instruções:

  1. Abra os arquivos HTML em qualquer editor de texto (Notepad, VS Code, Sublime Text, etc.)
  2. Localize as seções que deseja modificar, como títulos, textos ou links
  3. Faça as alterações desejadas, mantendo a estrutura HTML
  4. Salve o arquivo e atualize o navegador para ver as mudanças

Exemplo de edição de um título:

<h3>Demon Slayer: Nova Temporada</h3> <h3>Demon Slayer: Análise da Temporada 4</h3>

Como Adicionar Imagens

Para adicionar suas próprias imagens:

  1. Coloque as imagens na pasta images/
  2. Nos arquivos HTML, atualize os caminhos das imagens

Exemplo:

<img src="images/anime1.jpg" alt="Demon Slayer"> <img src="images/minha-imagem.jpg" alt="Minha Descrição">

Como Adicionar Novos Posts

Para adicionar um novo post:

  1. Duplique um arquivo de post existente (ex: post-anime1.html)
  2. Renomeie o arquivo (ex: post-anime7.html)
  3. Edite o conteúdo conforme necessário
  4. Adicione um link para o novo post na página da categoria correspondente

Como Modificar o Estilo

Para alterar cores, fontes e outros estilos:

  1. Abra o arquivo css/style.css
  2. Localize as propriedades que deseja modificar
  3. Altere os valores conforme desejado

Exemplo para mudar a cor do cabeçalho:

header { background-color: #1a1a2e; /* Cor atual */ color: white; } /* Altere para: */ header { background-color: #2a2a8e; /* Nova cor */ color: white; }

Visualização Local

Para visualizar o site localmente:

  1. Abra o arquivo index.html em qualquer navegador moderno
  2. Navegue pelo site usando os links do menu e os botões "Ver mais..."
  3. Após fazer alterações, salve os arquivos e atualize a página no navegador
